# Лабораторная работа №3 — Основы JavaScript

В рамках лабораторной работы были реализованы базовые функции на языке **JavaScript**, направленные на изучение синтаксиса, циклов, рекурсии, замыканий и работы с объектами.

## Реализованные функции
- `isInteger(n)` — проверка, является ли число целым (с использованием побитовых операторов)  
- `even()` — возвращает массив чётных чисел от 2 до 20  
- `sumTo(n)` и `recSumTo(n)` — сумма чисел до `n` (итеративно и рекурсивно)  
- `factorial(n)` — вычисление факториала числа  
- `isBinary(n)` — проверка, является ли число степенью двойки  
- `fibonacci(n)` — нахождение n-го числа Фибоначчи  
- `getOperationFn(initialValue, operatorFn)` — функция-замыкание для выполнения переданной операции  
- `sequence(start, step)` — генератор арифметической последовательности  
- `deepEqual(obj1, obj2)` — рекурсивное сравнение объектов по содержимому
